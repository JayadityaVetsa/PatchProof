# Support

PatchProof v0.1.0 is release-ready from source. Public registry publication and a hosted support channel are pending repository ownership setup.

After development begins:

- Use GitHub Discussions for usage questions and design conversation.
- Use GitHub Issues for reproducible defects and scoped feature requests.
- Use private vulnerability reporting for security concerns.
- Read existing documentation and search prior reports before opening a new thread.

Support is community-provided on a best-effort basis. No response-time or compatibility guarantee exists unless explicitly documented.

When asking for help, include the PatchProof version, operating system, Git version, language/runtime version, configuration with secrets removed, base/head revisions, command, and relevant diagnostics.

# Support

PatchProof is a public alpha provided on a best-effort basis.

- Use GitHub Issues for reproducible defects and focused feature requests.
- Use GitHub Discussions for usage questions and design conversation once Discussions are enabled.
- Use private vulnerability reporting for security concerns.

Include the PatchProof version, operating system, Git and runtime versions, redacted configuration, base/head revisions, command, and relevant diagnostics.

The supported alpha is the newest `0.1.x` prerelease. Compatibility claims are limited to the matrix documented in [docs/COMPATIBILITY.md](docs/COMPATIBILITY.md).

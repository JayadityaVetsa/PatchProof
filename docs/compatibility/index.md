# Compatibility

The alpha support target is:

- Node.js 22 and 24.
- Python 3.10 through 3.14.
- Linux, Windows, and macOS.
- pytest, Jest, and Vitest.
- npm, pnpm, Yarn, and Bun lockfile detection.
- One configured project per invocation.

Dynamic test generation, unusual monorepos, unsupported runners, historical dependency failures, and ambiguous output may require explicit configuration or become `inconclusive`.

# ADR-0002: JavaScript and Python adapters

- **Status:** Accepted
- **Date:** 2026-06-18

## Decision

JavaScript/TypeScript and Python receive equal v1 support through adapters over one language-neutral core.

## Rationale

They offer broad developer reach and distinct ecosystems that pressure-test the adapter boundary.

## Consequences

V1 requires parity fixtures and documentation for both. Framework support remains intentionally narrow and explicit rather than claiming universal compatibility.

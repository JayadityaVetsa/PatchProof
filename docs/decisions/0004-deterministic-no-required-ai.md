# ADR-0004: Deterministic core with no required AI

- **Status:** Accepted
- **Date:** 2026-06-18

## Decision

Canonical proof status is derived only from Git state and executed test evidence. No AI model, API key, or remote inference service is required.

## Rationale

Deterministic evidence is reproducible, private, inexpensive, and auditable.

## Consequences

Optional future AI may explain results but cannot alter canonical status. The product must avoid marketing itself as an AI reviewer.

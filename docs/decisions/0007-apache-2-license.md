# ADR-0007: Apache License 2.0

- **Status:** Accepted
- **Date:** 2026-06-18

## Decision

License PatchProof under Apache-2.0.

## Rationale

The license permits broad open-source and commercial use while providing an explicit patent grant.

## Consequences

Distributions and contributions must follow Apache-2.0 notice and attribution requirements. Third-party dependencies must be license-compatible.
